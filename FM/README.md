Fault Management
================

## Usage

```
$ git clone https://github.com/TJSoftwareReuse/2012T03.git
$ cd 2012T03
```

#### Install Gradle

(......Google or Baidu)

#### Gradle

```
$ gradle build
```

#### Configure Eclipse Project

1. Open your Ecipse
2. File -> New -> Java Project

    Remember to cancel 'Use default location' and change it to your project root directory
